# OpenLayers 定位导航组件规划

## 目标

新建 `ol-location-navigation`，实现一个可复用的 OpenLayers 定位导航组件库。第一版同时支持经纬度地图和室内图片地图，重点完成路网导入、最短路径规划、导航展示和基础编辑。

## 范围

- 地图模式：`geo` 经纬度地图、`image` 室内/图片地图。
- 数据模型：地图配置、楼层、节点、边、POI、路线结果。
- 路径算法：基于路网的 Dijkstra 最短路径，支持权重、禁行、单向边。
- 地图展示：节点、边、POI、规划路径、当前位置。
- 编辑能力：加节点、连线、拖动节点、导入导出 JSON。
- 适配器：Vue、React、Vanilla。

## 后续扩展

- 浏览器 GPS 实时跟踪。
- 偏航检测与重新规划。
- 多楼层切换面板。
- 更精细的转向提示和朝向箭头。
